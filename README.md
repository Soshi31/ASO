# ASO

Repository pentru temele ASO si aplicatia TODO.

Structura:

- `Tema1Si2/Tema1/` - tema 1, pentru identificarea serviciilor Windows care ruleaza in acest moment.
- `Tema1Si2/Tema2/` - tema 2 + tema optionala, care parcurge procesele active, verifica modulele DLL incarcate de fiecare proces si afiseaza DLL-urile folosite de mai multe procese, impreuna cu lista proceselor si numarul lor.
- `TemaOptionalaCitireFisier/` - tema optionala pentru citire fisier, care citeste de la input un path relativ, verifica daca fisierul exista si apoi afiseaza continutul lui.
- `Temaoptionalwikilocal/` - tema optionala wiki.local.
- `TodoApp/` - aplicatia TODO ASP.NET Core Razor Pages.
