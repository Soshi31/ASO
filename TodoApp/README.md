# ASO - TodoBasic

Aplicatie TODO simpla facuta cu ASP.NET Core Razor Pages.

Functionalitati:

- adaugare task nou
- marcare task ca finalizat/nefinalizat
- stergere task
- afisare total task-uri, task-uri finalizate si task-uri ramase

Datele sunt salvate local in `Data/todos.txt`, deci lista ramane disponibila dupa oprirea aplicatiei.

Pornire:

```powershell
dotnet run
```
